# 4. 誤解しやすい点・反例

1. **「ベクトル＝矢印」と思い込まない。** 関数も行列も多項式もベクトル。共通するのは「線形に組み合わせられる」という構造。

2. **次元と成分数を混同しない。** 「3 次元ベクトル空間」は基底の濃度が 3。基底を変えても次元は不変。

3. **成分は基底依存。** 「ベクトル $(1,0)$」と書いた時点で暗黙に標準基底を選んでいる。基底を変えれば同じベクトルの成分も変わる。

4. **ノルムは一意でない。** $L^1, L^2, L^\infty$ はすべて妥当なノルム。タスクに応じて選ぶ。

5. **線形独立と「ゼロでない」を混同しない。** 1 本のベクトルが線形独立 ⇔ そのベクトルが零ベクトルでない、という特殊ケースだけ一致する。一般には別概念。

6. **無限次元の罠.** 関数空間では基底の存在は自明ではない（選択公理が必要）。また「収束」を考えるとノルムだけでは不十分でバナッハ空間・ヒルベルト空間が必要になる。
