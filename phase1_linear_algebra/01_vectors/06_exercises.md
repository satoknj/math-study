## セクション 5: LLM/圏論との接続 ([[05_llm_category]])

### (c) 応用

- [ ] **Q10.** 単語埋め込みで類似度を測るとき、$L^2$ 距離 $\|\mathbf{v}-\mathbf{w}\|_2$ ではなく **コサイン類似度** $\frac{\langle \mathbf{v}, \mathbf{w}\rangle}{\|\mathbf{v}\|_2 \|\mathbf{w}\|_2}$ が好まれる理由を、ノルムと方向の観点から説明せよ。

---

<details>
<summary>解答</summary>

**A1.** (V7) $(a+b)\mathbf{v} = a\mathbf{v} + b\mathbf{v}$ はスカラーの和に関する分配律。(V8) $a(\mathbf{u}+\mathbf{v}) = a\mathbf{u} + a\mathbf{v}$ はベクトルの和に関する分配律。「何を分配するか」が違う。

**A2.** $p(x) = \sum a_i x^i$, $q(x) = \sum b_i x^i$ とおく。
- (V7): $(a+b)p(x) = \sum (a+b)a_i x^i = a p(x) + b p(x)$。
- (V8): $a(p(x)+q(x)) = a \sum (a_i + b_i) x^i = ap(x) + aq(x)$。
体 $\mathbb{R}$ の分配律から従う。

**A3.** $\forall c_1, \dots, c_k \in K,\; \left( \sum_{i=1}^k c_i \mathbf{v}_i = \mathbf{0} \implies \forall i,\; c_i = 0 \right)$

**A4.** $\mathbf{v}_1 + \mathbf{v}_2 = (1,1,2) = \mathbf{v}_3$。よって $\mathbf{v}_3 = \mathbf{v}_1 + \mathbf{v}_2$ で線形従属。

**A5.** **張ること:** 任意の $p \in P_2$ は $p(x) = a_0 + a_1 x + a_2 x^2$ と書ける。**線形独立性:** $c_0 + c_1 x + c_2 x^2 = 0$（恒等的に 0）とする。$x=0$ で $c_0=0$、微分して $x=0$ で $c_1=0$、再微分で $c_2=0$。$\blacksquare$

**A6.** (N1) 非負性・非退化性、(N2) 斉次性、(N3) 三角不等式。$L^2$ 以外: $L^1$（マンハッタン）、$L^\infty$（最大値）。

**A7.** $\|\mathbf{v}\|_1 = 1+2+2 = 5$、$\|\mathbf{v}\|_2 = \sqrt{1+4+4} = 3$、$\|\mathbf{v}\|_\infty = \max(1,2,2) = 2$。

**A8.** $\|\mathbf{v}\|_2 = \sqrt{9+0+16} = 5$。$\hat{\mathbf{v}} = (3/5,\; 0,\; -4/5)$。

**A9.** 証明の核心は「2次式 $\|\mathbf{v}+t\mathbf{w}\|_2^2 \ge 0$ の判別式 $\le 0$」。等号成立は判別式 $= 0$、すなわちある $t_0$ で $\mathbf{v} + t_0 \mathbf{w} = \mathbf{0}$。つまり **$\mathbf{v}$ と $\mathbf{w}$ が平行**（一方が他方のスカラー倍、$\mathbf{w}=\mathbf{0}$ も含む）。

**A10.** $L^2$ 距離はノルムの大きさにも依存する。コサイン類似度はノルムをキャンセルし「方向」のみを比較する。単語埋め込みでは頻出語ほどノルムが大きくなる傾向があるため、頻度ではなく意味（方向）で類似度を測りたいときコサイン類似度が適切。

</details>
