# 数学学習セッション

## 手順

### 1. 進捗確認

`ROADMAP.md` を読み、最初のチェックされていない単元（`[ ]`）を特定する。

### 2. 単元ファイルを開く

対応するファイルを読み込む。ファイルが存在しない場合は、単元の内容をその場で作成してファイルに保存する。

### 3. 概念説明

- 単元の概念を簡潔に説明する
- LLM・圏論との接続ポイントを示す

### 4. 確認問題（一問ずつ）

確認問題は **必ず一問ずつ** 出題する。全問を一度に出さない。

各問題のやり取り：
- **正解の場合**: 「正解です。」と伝え、次の問題へ進む
- **不正解・曖昧な場合**: 補足説明を加えて同じ問題を再出題する
- **「〜とは？」などの質問**: その場で説明し、同じ問題に戻る

### 5. 単元完了

全問正解したら：
1. `ROADMAP.md` の該当チェックボックスを `[ ]` → `[x]` に更新する
2. 次の単元を案内する
