# 数学学習セッション

## 手順

### 1. 進捗確認

1. `ROADMAP.md` を読み、最初のチェックされていない単元（`[ ]`）を特定する
2. その単元の `00_index.md` を読み、各セクションの状態（`[ ]` / `[/]` / `[x]`）を確認して途中状態を把握する
3. その日の学習ログ `study_logs/YYYY-MM-DD.md` がなければ作成する

### 2. 対象セクションの特定

`00_index.md` の学習の流れ表から、最初の未完了セクションを特定する。

- `[ ]` 未着手 → 手順3（解説）へ
- `[/]` 解説済み → 手順4（確認問題）へ
- `[x]` は完了済みなので次のセクションへ

### 3. 解説と質疑応答（未着手セクションのみ）

セクションファイルを読み込み、CLAUDE.md の「学習の深さに関する方針」に従って内容を説明する。

説明後、理解を確認する：
- 「何か質問はありますか？」と聞く
- 質問があれば CLAUDE.md の「ユーザーから質問を受けたときの対応」に従って対応する
- ユーザーが納得したことを確認したら、`00_index.md` の該当セクションを `[ ]` → `[/]` に更新し、手順4へ進む

### 4. 確認問題（一問ずつ）

`06_exercises.md` の対応セクションブロックから、未解答（`[ ]`）の問題を一問ずつ出題する。全問を一度に出さない。

各問題のやり取り：
- **正解の場合**: 「正解です。」と伝え、学習ログに記録してから次の問題へ進む
- **不正解・曖昧な場合**: 補足説明を加えて同じ問題を再出題し、学習ログに記録する
- **質問を受けた場合**: CLAUDE.md の「ユーザーから質問を受けたときの対応」に従う

### 5. セクション完了

セクションの確認問題を全問正解したら：
1. `00_index.md` の学習の流れ表で該当セクションを `[/]` → `[x]` に更新する
2. 全セクションが `[x]` になった場合のみ、`ROADMAP.md` の該当チェックボックスを `[x]` に更新する
3. 次のセクション（または次の単元）を案内する
